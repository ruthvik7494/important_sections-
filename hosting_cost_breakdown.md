# 📱 Important Clarifications

### ❗ Frontend Hosting
Mobile apps do NOT require traditional hosting like websites. Costs mainly involve **Development**, **App store publishing**, and **Updates**.

### ❗ Backend Costs
Backend is the **main cost driver** because APIs run continuously and handle all user traffic, orders, and payments.

---

# 🚀 Recommended Strategy

* **Phase 1**: Use Firebase / MongoDB (₹0 – ₹2,000/month)
* **Phase 2**: Move to AWS (EC2 + PostgreSQL) (₹10,000/month)
* **Phase 3**: Scale with Redis + Microservices (₹50,000+/month)
